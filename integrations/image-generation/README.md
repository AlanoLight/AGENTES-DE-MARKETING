# Image Generation Integration

Camada de integracao unificada para provedores de imagem:

- OpenAI
- Leonardo AI
- Canva

Implementacao ativa:
- `platform/api/src/integrations/image-generation`

Configuracao de chaves:
- `platform/api/.env` com:
  - `OPENAI_API_KEY`
  - `LEONARDO_API_KEY`
  - `CANVA_API_KEY`
  - `IMAGE_DEFAULT_PROVIDER`
