# Dashboard Local

Abra o arquivo [index.html](./index.html) no navegador para usar o cockpit local da Paulinhos Burguer.

## O que ele faz

- organiza os agentes por funcao
- mostra referencias prontas do nicho
- monta prompts para cada agente
- centraliza links, angulos e linhas de venda

## Como usar

1. selecione um agente na lateral
2. ajuste objetivo, produto, formato, oferta e canal
3. clique em `Gerar prompt`
4. use `Copiar prompt` e leve o texto para o chat do VS Code

## Observacao

Esta versao e local e estatica. Ela organiza e acelera o uso dos agentes, mas nao executa os agentes automaticamente sozinha.
